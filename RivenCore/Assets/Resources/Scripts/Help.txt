What do these folders mean?
General Terms
Framework Dev - A developer working on the Framework
Project Dev - A developer using the Framework to make a game

Asset Groups
Components - Assets added to the scene or to objects by project devs
Framework - Assets that provide core functionality to the framework
Framework Customs - Framework assets that may need customizing by project devs to fit current project needs
Integratables - Assets created by project devs that may be able to be added to the Framework
Project Specifics - Assets created for the current project with no intention to be added to the Framework or used in other projects
